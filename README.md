### Инструкции:
1. Клонируйте репозиторий на локальную машину:
~~~
git clone git@github.com:pestrige/export_link.git
~~~
2. Перейдите в папку проекта:
~~~
cd export_link
~~~
4. Установите зависимости:
~~~
yarn install
~~~
5. Запустите проект командой:
~~~
yarn start
~~~
